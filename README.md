# QUEST
QUEST, a computational method that quantifies uncertainty for spatial domain detection in spatial transcriptomics, generating probabilistic domain assignments and location-level uncertainty estimates. QUEST is implemented as an open-source R package with efficient C++ codes linked through Rcpp, freely available at www.xzlab.org/software.html.
